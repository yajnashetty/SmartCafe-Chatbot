# SmartCafe-Chatbot

☕ CafeBot: AI-Powered Cafe Ordering Assistant
CafeBot is a simple AI-powered chatbot built using Flask that allows users to interactively place orders at a cafe. The bot responds to menu inquiries, processes orders, calculates bills, and provides a smooth conversational experience.

🚀 Features
🗨️ Interactive chatbot for cafe orders

📜 Displays the full menu with pricing

🛒 Adds items to the order dynamically

🧾 Generates the total bill based on selected items

🔄 Allows users to clear and restart orders

🎨 Simple and clean web UI using HTML, CSS, JavaScript

🛠️ Tech Stack
Backend: Flask (Python)

Frontend: HTML, CSS, JavaScript

Data Handling: Flask session for order tracking
